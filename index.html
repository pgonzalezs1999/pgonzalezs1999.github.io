Proyectos

WeFood

Aplicación completa para Android / iOS para vender a precio reducido comida no vendida por restaurantes y hoteles durante una jornada, para reducir el desperdicio de alimentos. Desarrollada al completo, incluyendo registro de usuarios, inicio de sesión, reconocimiento de huella dactilar y FaceID, base de datos, API, pasarela de pagos para latinoamérica y conexión a la API de Google Maps. Abandonada por falta de presupuesto e incapacidad de captar clientes reales. Deslistada de los buscadores por inactividad. Código en GitHub

ReadyRecipes

Buscador de recetas online en base a múltiples parámetros, como ingredientes, tipo de comida, hora del día... conectada a una base de datos poblada webscrapeando videos de YouTube y procesadas automáticamente mediante una inteligencia artificial entrenada por nosotros. Proyecto de universidad
